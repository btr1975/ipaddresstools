3. Multicast ip, and mask

5. ip, and mask
6. ip
7. Correct neighbor ip
8. Create a subnet form a ip, and mask combo
9. Verify CIDR
10. Get subnet range
11. Get all subnets possible
12. Get all longer subnets possible
13. Get all shorter subnets possible
